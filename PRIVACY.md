# Privacy Policy

**PR Zen** is a browser extension that helps developers navigate noisy GitHub Pull Requests.

## Data Collection

PR Zen does **not** collect, transmit, or store any personal data. It does not communicate with any external server.

## Local Storage

PR Zen uses `chrome.storage.sync` to persist your preferences (enabled state, bot patterns, alert keywords, minimum bot comment threshold). This data is stored locally in your browser and synced across your Chrome instances via your Google account. PR Zen does not have access to your Google account.

## Page Access

PR Zen runs exclusively on `github.com` pull request pages. It reads the page DOM to classify comments as human, bot, or alert. This analysis happens entirely in your browser — no page content is sent anywhere.

## Third-Party Services

PR Zen does not use analytics, tracking, ads, or any third-party service.

## Changes

If this policy changes, the update will be published in this repository before the new version is released on the Chrome Web Store.
